# Freecommerce

An ecommerce from scratch site with modern tech stack:

- nx
- react
- nextjs
- tailwind
- daisyui
- typescript

```
git clone https://github.com/hanspoo/freecommerce
cd freecommerce
```

Levantar redis

```
cd docker/
docker compose -f redis.yml up
```

En otro terminal ejecutar app

```
npm install
npm run dev
```

Ir al navegador:

http://localhost:3000
